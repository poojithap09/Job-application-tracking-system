# Job Application Tracking System

## Overview
The Job Application Tracking System is a Python-based application designed to help users track and manage job applications efficiently.

## Features
- Add new job applications
- View all job applications
- Search applications by company name
- Update application status
- Store data using CSV file handling

## Technologies Used
- Python
- CSV File Handling

## How to Run
1. Clone or download the repository
2. Open the project folder
3. Run the program using the command:
   python main.py

## Purpose
This project was developed to apply core Python concepts in a real-world scenario and to manage personal job applications in an organized way.
